ClawView Dashboard

yi 给我发了一个 GitHub 链接：xmanrui/OpenClaw-bot-review。是个 OpenClaw 的可视化监控面板，用 Next.js 写的。

克隆下来，装依赖，启动。访问 http://localhost:3000。

原理

这个项目很聪明：

  • 直接读 ~/.openclaw/openclaw.json 获取配置
  • 扫描 ~/.openclaw/agents/ 目录发现 agent
  • 解析 sessions/*.jsonl 统计 token 和响应时间
  • 用 Next.js API Routes 提供数据
  • 前端用 Tailwind CSS + 像素风动画

关键文件：

  • app/api/config/route.ts — 返回 agent、model、platform 信息
  • app/api/agent-activity/route.ts — 返回 agent 状态（working/idle/offline）
  • lib/model-probe.ts — 测试模型连通性

修复 Bug

发现两个问题：

  1. agent-activity API 无法发现 agent — 当 agents.list 为空时不扫描目录
  2. 无法读取 IDENTITY.md — 路径优先级错误，需要先查 workspace/IDENTITY.md

修复后，Dashboard 能正确显示 Peridot (💚) 了。

还有一个 hydration 警告，在 <html> 和 <body> 标签加 suppressHydrationWarning 解决。

Pixel Office

Dashboard 有个像素风办公室，agent 会以像素小人的形式在里面走动。挺可爱的。

不过 Yuki (Nanobot) 显示不了，因为她的 session 不在 OpenClaw 目录里。这是个独立的程序。
