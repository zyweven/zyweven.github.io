4. Sort an Array（排序数组）✅ 已 AC

难度： 中等
通过率： 47.8%（874K 提交）
要求： 不用内置函数，O(n log n)，空间尽可能小

题目

约束分析

  • 不能用 Arrays.sort() / sort.Ints() / sort() 等内置排序
  • 时间 O(n log n)：可用归并排序、堆排序、快速排序

面试常问：三种排序对比

排序平均时间最坏时间空间稳定

快速排序O(n log n)O(n²)O(1)❌ 不稳定
堆排序O(n log n)O(n log n)O(1)❌ 不稳定

面试建议： 写归并排序最稳（必问分治思想），可以提到快排和堆排序的 trade-off。

  • 归并排序：分治思想，先递归排序左右两半，再 merge

  • 稳定排序的意义：多键排序场景，不打乱之前排好的顺序

9. 相交链表 ✅ 已 AC

难度： 简单
通过率： 58.3%（1.8M 提交）
同款题： 剑指 Offer II 023. 两个链表的第一个相交节点

题目

给你两个单链表的头节点 headA 和 headB，判断两个链表是否相交。如果相交，返回相交的起始节点。要求 O(n) 时间，O(1) 空间。

核心思路

方法一：哈希集合（O(n) 时间，O(n) 空间）
遍历 A 入 set，再遍历 B，第一个在 set 中的就是交点。

方法二：双指针（最优，O(1) 空间）
关键结论：a + c = b + c（A 未相交部分 + 公共尾巴 = B 未相交部分 + 公共尾巴）

Java 解法（双指针，O(1) 空间）

ACM 模式（Java）

输入：

输出： 2

Go 解法（双指针）

C++ 解法（双指针）

ACM 模式（C++）

关键点

  • 双指针核心：A 走完走 B，B 走完走 A，路径长度相等（a + c = b + c）
  • O(1) 空间：不用哈希集合，面试加分
  • 相交判定：第一次相等时就是交点（或同时为 null）
  • 扩展：如果需要返回第一个相交节点而不是判断相交，同理

13. 环形链表 ✅ 已 AC

题目

给定一个链表，判断链表中是否有环。
如果链表中有某个节点，可以通过连续跟踪 next 指针再次到达，则链表中存在环。

LeetCode: 141. 环形链表

核心思路

Floyd 快慢指针检测环。

  • slow 一次走一步，fast 一次走两步
  • 如果有环：fast 最终会追上 slow（相遇）
  • 如果无环：fast 先到终点（null）

为什么有环一定会相遇？
设环长 C，fast 比 slow 多走一节。slow 进环时，fast 领先不到一圈，两指针相对速度为 1，每次距离减 1，最多减 C 次就相遇。

时间 O(n)，空间 O(1)——最优解。

Java 解法（快慢指针）

Go 解法（快慢指针）

C++ 解法（快慢指针）

追问扩展

① 如果要返回环的入口节点（142. 环形链表 II）怎么办？

设链表起点到环入口距离为 a，环长 C。相遇时 slow 走了 a+x，fast 走了 a+x+kC（k≥1）。

fast 走的距离是 slow 的两倍：2(a+x) = a+x+kC → a = kC - x

这意味着：从相遇点走到入口的距离 = 从起点走到入口的距离。

所以相遇后，让一个指针从头出发，一个从相遇点出发，同速前进，再次相遇就是入口。

② 哈希表法（另一种思路）

用 HashSet 记录访问过的节点，第一次重复就是环入口。时间 O(n)，空间 O(n)，但不如快慢指针优雅。

③ 为什么快指针不走三步？

可以，但没必要。慢指针每轮相对快指针多走一步，相对距离最多 n 步，两步一定能追上。三步最坏情况要走 2n 步，效率一样但实现更复杂。

关键点

  • 快慢指针模板：while (fast != null && fast.next != null)
  • 循环内：slow 一步，fast 两步
  • 相遇 → 有环；fast 到 null → 无环
  • 无环链表 O(1) 空间，这是快慢指针的经典应用

易错点

  • 别忘了 fast.next != null 判断，否则 fast.next.next 可能 NPE
  • 不要改变原链表结构（不破坏链表再判断）

题目

给定一个非负整数 numRows，生成杨辉三角的前 numRows 行。

核心思路

杨辉三角每行首位和末位是 1，中间元素 = 上行同列 + 上行前一列。

规律：triangle[i][j] = triangle[i-1][j] + triangle[i-1][j-1]

追问扩展

① 杨辉三角第 n 行第 k 个元素（不用建整个三角）？

直接用组合公式：C(n-1, k-1)。第 n 行（从 1 开始）第 k 个 = C(n-1, k-1)。

② 杨辉三角所有行都是回文？

是的，每行从左到右和从右到左一样。

③ 杨辉三角的物理意义？

(a + b)^n 展开式的系数就是第 n 行。二项式定理的直观体现。

关键点

  • triangle[i][j] = triangle[i-1][j-1] + triangle[i-1][j]
  • 每行首尾必然是 1，单独处理
  • 建整个三角 O(n²)，但第 n 行有 n 个元素，无法更好

易错点

  • 第 i 行有 i+1 个元素（不是 i 个）
  • 内层循环 j 从 1 到 i-1，首尾不用算
